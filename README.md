## 프로젝트 소개

본 프로젝트는 소프트웨어공학 수업 중 웹 애플리케이션 구현 프로젝트의 최종 단계에 해당하는 과제로 Github를 통한 공동 개발 환경 하에 이전 프로젝트 단계에서 설계한 학사 관리 시스템의 구현을 목표로 한다.
  
본 과제의 학사 관리 시스템의 requirement는 8가지의 기본 기능과 4가지의 추가 기능으로, 우리 9조에서는 총 13가지 기능 구현에 주력하였다.

## 고찰

본 프로젝트를 통해 최종적으로 학기 중 목표로하던 학사 관리 시스템 웹 애플리케이션의 구현을 완성하였다. 프론트엔드와 백엔드로 나뉘어서 작업하였으며 이를 통해 실제 공동 개발 환경에서의 작업을 경험해 볼 수 있었다. 
인적 자원이 부족해 프로젝트의 스케줄 관리 및 팀원간 역할 분배에 있어 난향을 겪기도 했지만 이를 바탕삼아 공동 개발 환경에서의 조율 능력을 키울 수 있었다. 한편, 프론트엔드와 백엔드의 작업 과정에서 관련된 여러 웹 기술들에 대해서도 접하며 익힐 수 있었다. 
학사 관리 시스템 프로젝트를 통해 실제 현장에서의 공동 개발 환경과 유사한 환경에서의 개발 경험을 통해 프로그램 개발자로서 역량을 키울 수 있었다.
